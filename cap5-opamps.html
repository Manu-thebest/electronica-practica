5.1 Regas de Oro del OpAmp Ideal

Estas reglas simplifican el análisis de cualquier circuito con operacionales:

  1. Impedancia de entrada infinita: No fluye corriente hacia los terminales + ni -
  2. Impedancia de salida cero: Puede entregar corriente sin caer el voltaje
  3. Ganancia infinita en lazo abierto: Sin retroalimentación, la salida satura
  4. Cortocircuito virtual: $V_+ = V_-$ cuando hay retroalimentación negativa

5.2 Amplificador Inversor

$\displaystyle A_v = -\frac{R_f}{R_{in}}$
Ganancia negativa = salida invertida 180°
💡 Diseño rápido $R_f = 10\text{k}\Omega$, $R_{in} = 1\text{k}\Omega$ → Ganancia = $-10$ (amplifica 10× e invierte).
Corriente de entrada: $I_{in} = V_{in}/R_{in}$. Si $V_{in}=1V$, $I_{in}=1mA$.

5.3 Amplificador No-Inversor

$\displaystyle A_v = 1 + \frac{R_f}{R_1}$
Ganancia ≥ 1, salida en fase con entrada
💡 Casos útiles • $R_f = 0$ → $A_v = 1$ (buffer/seguidor)
• $R_f = R_1$ → $A_v = 2$
• $R_f = 9R_1$ → $A_v = 10$

5.4 Seguidor de Voltaje (Buffer)

$\displaystyle V_{out} = V_{in}$
Conecta salida directamente al terminal inversor ($R_f=0$, $R_1=\infty$)
🎯 ¿Para qué sirve? Separa un sensor de alta impedancia (termopar, micrófono electret) de una carga que consume corriente. Ganancia de corriente ×1000 o más, aunque el voltaje no crezca.

5.5 Amplificador Sumador

$\displaystyle V_{out} = -R_f\left(\frac{V_1}{R_1}+\frac{V_2}{R_2}+\frac{V_3}{R_3}\right)$

Si todas las R son iguales: $V_{out} = -(V_1+V_2+V_3)$. ¡Suma inversora!

5.6 Amplificador Diferencial

$\displaystyle V_{out} = \frac{R_f}{R_1}(V_2-V_1) \quad$ si $\frac{R_1}{R_f}=\frac{R_2}{R_g}$
💡 ¿Cuándo usarla? Cuando necesitas medir la diferencia entre dos señales con ruido común. Puente de Wheatstone, sensores de corriente, audio balanceado.

5.8 Trigger de Schmitt

Agrega histéresis a un comparador para falsas conmutaciones por ruido:

$\displaystyle V_{TH+} = V_{ref}\frac{R_2}{R_1+R_2}+V_{OH}\frac{R_1}{R_1+R_2}$
$\displaystyle V_{TH-} = V_{ref}\frac{R_2}{R_1+R_2}-V_{OL}\frac{R_1}{R_1+R_2}$
� Histeresis $V_H = V_{TH+} - V_{TH-}$. Cuanto mayor, más inmune al ruido. Típico: 50-200mV.

5.R Resumen y Errores Comunes

FórmulaUso
$A_v = -R_f/R_{in}$Inversor
$A_v = 1+R_f/R_1$No-inversor
$V_{out} = -R_f\sum(V_i/R_i)$Sumador
$V_{TH\pm} = V_{ref}\frac{R_2}{R_1+R_2} \pm V_{O}\frac{R_1}{R_1+R_2}$Schmitt trigger
⚠️ Errores comunes
  1. Olvidar la alimentación del OpAmp (±Vcc)
  2. Conectar señal al terminal equivocado (inversor vs no-inversor)
  3. Usar un OpAmp lento para señales rápidas
  4. No calcular el ancho de banda: GBW = ganancia × fmax
  5. Ignorar el límite de slew rate (V/µs)
